CIRCULAR WEATHER

An interactive data visualization using JavaScript and open source P5.js. Took advantage of an open weather live API and extracted data for the temperature humidity and wind levels for 3 days in the row and visualized it with rings that their color pallet changes depending how high or low the 3 parameters are. This project was great practice on how to handle big datasets and a simple but comprehensible way of visualizing the information.
